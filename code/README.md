## Model Overview

A **3-class kinetic traffic model** on a **periodic (ring) road**, simulated with **Lie–Trotter 3-phase operator splitting**:

| Class | Meaning | PCE weight |
|-------|---------|-----------|
| **A** | Trucks (moving bottleneck) | 2.5 |
| **Bf** | Free-flowing cars | 1.0 |
| **Bs** | Trapped cars (stuck behind trucks) | 1.0 |

Each class has a distribution over **N = 15 discrete speed classes** (v = 2, 4, … 30 m/s).

### 3-Phase Time Step

```
Phase 1  Capture & Release  (exact 2×2 matrix exponential, per cell)
         σ = σ_0 · P_block · A^(i)                    [Eq. 10, no B(Ω_x)]
         μ = μ_0 · exp(-P_block · θ_A / R_A)          [Eq. 12]
         f_Bf, f_Bs exchange via exact matrix exp

Phase 2  Kinematics         (Thomas tridiagonal solve)
         Deceleration λ_{i→i-1}  uses  local Ω_x     [Eq. 5]
         Acceleration λ_{i→i+1}  uses  downstream Ω_{x+1}  [Eq. 4]  ← asymmetric
         Bs acceleration blocked above i_thr

Phase 3  Advection          (global Godunov limiter, ring BC)
         Ψ = v_i · f_{i,x}^{(m)}                      [Eq. 13, pure demand]
         α = min(1, (ρ_max − Ω_{x+1} + ε) / (D + ε)) [Eq. 15]
         Φ = α · Ψ                                    [Eq. 16]
```

### Key Design Decisions

| Decision | Rationale |
|----------|-----------|
| **Acceleration uses Ω_{x+1}** (not local Ω_x) | Trucks' own PCE=2.5 inflates Ω_x and self-traps them. Downstream look-ahead fixes this. |
| **σ without B(Ω_x)** singular barrier | P_block already captures congestion; B would double-count. Removed to keep Phase 1 bounded and smooth. |
| **Ψ = v·f (pure demand)** | Supply filter in Ψ causes double-dipping with α limiter. Cleaner to use pure demand + α. |
| **ε_tol in both α numerator and denominator** | Prevents micro-flux suffocation when both demand and space are near zero. |
| **Ring road (periodic BC)** | Clean closed system — total mass is exactly conserved. |

## Initial Conditions (Main Benchmark)

Identical across `generate_dataset.py` and `generate_dataset_LWR.py`:

| Region | Class | Density | Speed class | Physical speed |
|--------|-------|---------|-------------|----------------|
| cells 0–73 (upstream) | Bf | 0.020 | i = 14 | 30 m/s |
| cells 74–79 (bottleneck) | A (trucks) | 0.040 | i = 6 | 14 m/s (truck free-flow) |
| cells 80–149 (downstream) | — | 0 (empty) | — | — |
| all cells | Bs | 0 | — | — |

Notes:
- Initial PCE occupancy in truck zone: **Ω = 2.5 × 0.040 = 0.100 ≈ 67% ρ_max** — meaningful bottleneck without sealing the road.
- Bs must be generated by Phase 1 dynamics; it cannot be seeded.
- The downstream empty region is required for clean rarefaction wave validation (V3-g).

## Core Equations

### Occupancy (Eq. 1)

$$\Omega_{x,l} = \sum_{m} w^{(m)} \sum_i f_{i,x,l}^{(m)}$$

### Asymmetric Kinematic Transitions (Eq. 4 & 5)

$$\lambda_{i\to i+1}^{(m)} = a_i^{(m)}\!\left(1 - \frac{v_i}{v_{\max}}\right)^{\delta^{(m)}}
\left[1 - \exp\!\left(-\frac{\max(0, \rho_{\max} - \Omega_{x+1})}{R_c}\right)\right]$$

$$\lambda_{i\to i-1}^{(m)} = \text{(local Ω}_x\text{-based deceleration with singular barrier }\mathcal{B}(\Omega_x)\text{)}$$

### Probabilistic Blocking (Eq. 8)

$$P_{\text{block}}(x) = \left(\frac{\Omega_x}{\rho_{\max}}\right)^{\eta_{\text{block}}}, \quad \eta_{\text{block}} = 2$$

### Capture Rate (Eq. 10) — **no singular barrier**

$$\sigma_x^{(i)} = \sigma_0^{(B)}\, P_{\text{block}}(x)\, A_x^{(i)}$$

### Pure Demand Flux (Eq. 13)

$$\Psi_{i,x\to x+1}^{(m)} = v_i\, f_{i,x}^{(m)}$$

### Godunov Limiter (Eq. 15)

$$\alpha_{x\to x+1} = \min\!\left[1,\; \frac{\max(0,\, \rho_{\max} - \Omega_{x+1}) + \varepsilon_{\text{tol}}}{\mathcal{D}_{x\to x+1} + \varepsilon_{\text{tol}}}\right]$$

## Scripts

### `generate_dataset.py` — Main kinetic benchmark
Full 3-phase kinetic model, produces `multiclass_trm_vA.h5` (then copied to `multiclass_trm_benchmark_500mb.h5` for validation pipeline).

### `generate_dataset_nobot.py` — No-bottleneck baseline
Pure Riemann problem (sparse left, dense jam right, no trucks). Tests Phase 2 + 3 in isolation. Outputs `multiclass_trm_nobot.h5` and `figures/V_nobot_hovmoller.png`.

**IC**: sparse Bf (ρ=0.005, v=30 m/s) on cells 0–74 | dense jam (ρ=0.120, v=2 m/s) on cells 75–149. T = 100 s (200 steps).

### `generate_dataset_LWR.py` — First-order LWR variant
Same initial condition as main benchmark, but replaces Phase 2 + Phase 3 with an instantaneous-speed triangular fundamental diagram. **Phase 1 is preserved identically**, allowing direct isolation of multi-class transition dynamics from kinetic speed dynamics. Outputs `multiclass_trm_LWR.h5` and `figures/v_lwr.png` (2×3 comparison).

**LWR parameters**: U = 14 m/s (truck), V = 30 m/s (car), W = 4.62 m/s (wave), ρ_c ≈ 0.020, Q_cap ≈ 0.60 PCE/m/s. Capacity allocation is density-proportional; supply uses unweighted ρ_total.

### `run_all.py` — Validation pipeline
Runs V1–V7 modules in sequence, aggregates results, writes `validation_summary.json`.

## Riemann Problem Validation

Two canonical Riemann problems are encoded in the main benchmark.

### Shock Wave (V3-c) — free flow hits bottleneck
Upstream Bf at v=30 m/s collides with truck platoon. Density jump propagates **backward** at ≈ −4.62 m/s (Rankine-Hugoniot). Check: upstream Ω increases from t=0 to late time, and ρ_Bs > 0.001 confirms capture at the shock.

### Rarefaction Wave (V3-g) — downstream discharge
Traffic escapes from the bottleneck into empty space. Density fan expands **forward** smoothly. Check: max cell-to-cell gradient in x ∈ [80, 110] is < 0.5 ρ_max; far-downstream ρ_Bs << bottleneck ρ_Bs.

## Stiffness Numbers

| Quantity | Value |
|----------|-------|
| τ_adv = Δx / v_max | 0.667 s |
| τ_relax = 1 / λ_dec,max | down to ~10⁻¹⁵ s near Ω = ρ_max |
| Time-scale separation | up to ~10¹⁵ |
| Initial stiffness ratio S(t=0) | ~10⁵ |
| Peak stiffness ratio S_max | ~10¹⁵ |
| Thomas algorithm residual | ~10⁻¹⁸ |

Explicit Euler diverges at this stiffness; Phase 1 uses the **exact matrix exponential**, Phase 2 uses **Thomas tridiagonal**, Phase 3 is **explicit** under CFL = 0.75.

## Historical Note

Earlier versions (m3+m4, m7) used a 4-phase splitting with explicit lateral lane-changing (Softplus γ, escape gate G, entrapment E_trap). That lateral phase was removed and replaced with `P_block` — an algebraic probabilistic blocking factor — giving the current 3-phase structure with independent lanes. See commit history for the m3+m4 → m7 → vA transition.
