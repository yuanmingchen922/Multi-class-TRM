## 1. 模型核心变化：从 Softplus 变道到概率阻断

### 旧版 vs 新版架构对比

| 对比项 | 旧版（Softplus变道） | 新版（P_block概率阻断） |
|--------|---------------------|------------------------|
| 算子分裂 | **4相**：捕获释放->运动学->侧向变道->对流 | **3相**：捕获释放->运动学->对流 |
| 侧向变道 | Softplus 函数，逃逸门 G，困陷因子 E_trap | **完全移除** |
| 核心新机制 | 无 | 概率阻断因子 P_block(x) |
| 车道耦合 | 车道间相互影响 | 各车道作为独立1D系统求解 |
| 关键参数 | eta_lat, xi, R_gap, omega_sp, kappa_hz | **eta_block**（概率阻断指数） |
| HDF5 数据 | gamma_left, gamma_right, E_trap | **P_block** |

### 新版核心公式

**概率阻断因子（Eq. 8）：**

    P_block(x) = (clip(Omega_x, 0, rho_max) / rho_max) ^ eta_block

- Omega_x = 0（空旷路段）-> P_block = 0，无阻断，捕获率趋零
- Omega_x = rho_max（堵死路段）-> P_block = 1，完全阻断，最大捕获率
- eta_block = 2.0，关于密度单调递增

**捕获率（Eq. 9）：**

    sigma_x^(i) = sigma_0 * P_block(x) * A_x^(i) * B(Omega_x)

**有效瓶颈压力（Eq. 10-11）：**

    theta_A = sum_{k<=i_thr} w^(A) * f_{k,x}^(A) / rho_max
    S_tilde = P_block(x) * theta_A

**释放率（Eq. 12）：**

    mu_x = mu_0 * exp(-S_tilde_x / R_A)

## 4. 代码逻辑说明

### 三相算子分裂（Lie-Trotter）

```
每个时间步 Delta_t：

Phase 1: 捕获/释放（精确矩阵指数）
  omega -> P_block = (omega/rho_max)^eta_block
  sigma = sigma_0 * P_block * A^(i) * B(omega)
  mu = mu_0 * exp(-P_block * theta_A / R_A)
  f_Bf* = f_Bf * exp(-S*dt) + mu * F * dt * phi(S*dt)
  f_Bs* = F - f_Bf*

Phase 2: 运动学（代数投影 + Thomas算法）
  投影：f^(Bs)[i > kappa*] -> f^(Bs)[kappa*]
  lambda_acc^(Bs)[i >= i_thr] = 0（加速封锁）
  (I - Dt*A) f_new = f_old（隐式Thomas）

Phase 3: 空间对流（全局 Godunov 限制器）
  Psi = v_i * f * supply_filter
  D = (Dt/Dx) * sum_m w_m * Psi（全局聚合需求）
  alpha = min(1, available/D)（全局限制器）
  Phi = alpha * Psi
  f += (Dt/Dx) * (Phi_in - Phi_out)
```

### 为什么用 P_block 而不是 E_trap？

**旧版 E_trap** 需要访问相邻车道的密度信息，造成：
1. 车道间耦合（每条车道不再独立）
2. 边界条件复杂（需要特殊处理最外侧车道）
3. 引入额外参数 xi（跨道感知权重）

**新版 P_block** 仅使用本格子的密度 Omega_x：
1. 各车道完全独立，可并行求解
2. 物理意义更清晰：局部密度直接决定阻断概率
3. 减少参数：去除 eta_lat, xi, R_gap, omega_sp, kappa_hz

### 黎曼问题详解与数值验证

本基准场景内嵌两种典型黎曼问题，各自通过显式定量检验进行验证。

#### 黎曼问题 I：激波（上游自由流 → 下游拥堵）

**场景设置**：高速 Bf 车群（x=59-69，速度30m/s）接近静止卡车瓶颈（x=74-79，Omega ≈ rho_max）。

**物理机制**：
- 在瓶颈上游界面处，Rankine-Hugoniot 条件形成密度跳跃
- Omega → rho_max 触发 B(Omega) → ∞（极端刚性激活）和 P_block → 1
- sigma → 最大值：Bf 被快速捕获为 Bs
- 密度间断面向上游传播，形成经典交通激波

**数值验证——检验 V3-c**：
- 量化指标：omega(x=68-73) 在 t=0 与 t=50s 的对比
- 正向增量（上游密度增大）确认激波从瓶颈向上游自由流区域传播
- 辅助信号：rho_Bs > 0.001 确认捕获事件在激波锋面处发生

#### 黎曼问题 II：稀疏波（上游拥堵 → 下游自由流）

**场景设置**：瓶颈下游（x=80+）初始为稀疏或空旷路段。瓶颈处拥堵车辆缓慢逃入此自由流区域。

**物理机制**：
- 下游 Omega 远低于 rho_max → P_block 趋小 → mu 增大
- Bs 被困车缓慢释放为 Bf 自由车
- 密度以扩展扇形式平滑递减，无尖锐锋面——这正是稀疏波（而非激波）的特征
- 扩展波向前传递"前方路段畅通"的交通信息

**数值验证——检验 V3-g**：
- 量化指标：在 x ∈ [80,110] 范围内，相邻格子密度梯度最大值 |omega(x+1) - omega(x)|
- 平滑梯度 < 0.5*rho_max 确认下游无激波形成（稀疏扇，而非新激波）
- 辅助信号：rho_Bs(远下游) << rho_Bs(瓶颈区) 确认稀疏波区域交通处于自由流状态（无捕获发生）
